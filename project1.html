System Engineering Project

The 2D Printer Pen

The 2D printer pen is capable of printing a wide range of high-resolution images and designs directly onto beverages. These pens are intended to be used by cafes and restaurants to attract more customers. Currently only those without disabilities are able to use this pen.

This project aims to reach out to more people that can use the pen by catering it to the disabled with hand tremors.
